Class: Region

westures-core/Region~Region(element, capture, preventDefault)

Allows the user to specify the control region which will listen for user input events.

Methods

activate()

Activates the region by adding event listeners for all appropriate input events to the region's element.

arbitrate(event)

All input events flow through this function. It makes sure that the input state is maintained, determines which bindings to analyze based on the initial position of the inputs, calls the relevant gesture hooks, and dispatches gesture data.
Parameters:
Name Type Description
