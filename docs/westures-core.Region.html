    src/Region.js, line 37

    Constructor function for the Region class.
    Parameters:
    Name Type Default Description
    element Element The element which should listen to input events.
    capture boolean false Whether the region uses the capture phase of input events. If false, uses the bubbling phase.
    preventDefault boolean true Whether the default browser functionality should be disabled. This option should most likely be ignored. Here there by dragons if set to false.
