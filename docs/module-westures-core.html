Module: westures-core

The global API interface for Westures. Exposes a constructor for the Region and the generic Gesture class for user gestures to implement, as well as the Point2D class, which may be useful.
Properties:
Name Type Description
Region Class The "entry point" class for Westures. Gets the ball rolling, so to speak, when it is instantiated.
Gesture Class This class is the one all Gestures using this library should extend.
Point2D Class This class is available for convenience. It defines some basic operations on a point in a two dimensional space.
